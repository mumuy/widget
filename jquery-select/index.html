$(".select1").select({ 'onSelect':function(item){$('#test').text('选中选项：'+item['name']);} });

调用方法：

$(selector).select(options,callback(api));

options参数

参数 默认值 说明
prefix 'widget' 自定义节点class的前缀
format(item) function(item){ return '<span>'+item['name']+'</span>'; } 下拉菜单自定义结构。item['name']选项名称,item['value']选项值。
onSelect(item) [无] 选择时触发。item['name']选项名称,item['value']选项值。

callback(api)参数

方法 说明
setValue() 修改当前值
