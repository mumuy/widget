### 调用
统一的调用方式：
```javascript
$(selector).widgetName(options,callback(api));
```
---

简介：支持xy双方向、无缝滚动、响应式、移动端手势操作、十余种动画特效等等
[演示](http://jquerywidget.com/jquery-slider/)
